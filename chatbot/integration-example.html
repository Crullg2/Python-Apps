VA Healthcare AI Chatbot - Integration Example

This page demonstrates how to integrate the VA Healthcare AI Chatbot into your website with minimal setup.

Step 1: Include Dependencies

Add Font Awesome for icons and include the chatbot script:

<link href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css" rel="stylesheet"> <script src="va-chatbot.js"></script>

Step 2: Add HTML Structure

Include the required DOM elements with proper IDs:

<div class="chatbot-container"> <div class="chatbot-header"> <button id="speakerButton"><i class="fas fa-volume-up"></i></button> <h2>VA Healthcare Assistant</h2> </div> <div id="chatMessages" class="chat-messages"></div> <div id="typingIndicator" class="typing-indicator"></div> <div class="chat-input-container"> <div class="chat-input-wrapper"> <button id="voiceButton" class="voice-button"><i class="fas fa-microphone"></i></button> <input type="text" id="chatInput" class="chat-input" placeholder="Ask a question..."> <button id="sendButton" class="send-button"><i class="fas fa-paper-plane"></i></button> </div> <div id="voiceStatus" class="voice-status"></div> <div id="errorMessage" class="error-message"></div> </div> </div>

Step 3: Initialize Chatbot

Create a new chatbot instance:

<script> // Initialize the chatbot when page loads document.addEventListener('DOMContentLoaded', function() { const chatbot = new VAChatbot(); console.log('VA Healthcare Chatbot initialized!'); }); </script>

Key Features

  • Context Awareness: Remembers conversation history
  • Follow-up Questions: Handles "How is it done?" and similar
  • Voice Support: Speech-to-text and text-to-speech
  • Training Integration: Can learn from uploaded documents
  • Simple Language: Automatically explains medical terms
  • Mobile Responsive: Works on all devices
  • Accessibility: Screen reader and keyboard navigation support

Customization

Customize the appearance with CSS variables:

:root { --va-primary: #003f7f; --va-secondary: #0078d4; --va-accent: #ffc72c; }
